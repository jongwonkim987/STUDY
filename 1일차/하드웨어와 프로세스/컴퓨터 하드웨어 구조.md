# 컴퓨터 하드웨어 구조

## 주요 구성 요소

### 1. CPU (Central Processing Unit)
- 컴퓨터의 두뇌 역할
- 명령어를 해석하고 연산을 수행
- 구성: ALU(연산장치), CU(제어장치), 레지스터

### 2. 메모리 (Memory)
- **RAM**: 실행 중인 프로그램과 데이터를 임시 저장 (휘발성)
- **ROM**: 읽기 전용 메모리, 부팅 정보 저장 (비휘발성)
- **캐시(Cache)**: CPU와 RAM 사이의 고속 임시 메모리

### 3. 저장 장치 (Storage)
- HDD, SSD 등
- 데이터를 영구적으로 저장 (비휘발성)

### 4. 입출력 장치 (I/O Devices)
- 입력: 키보드, 마우스
- 출력: 모니터, 프린터

## 버스(Bus) 구조
```
CPU ←→ 메모리 버스 ←→ RAM
 ↕
I/O 버스
 ↕
입출력 장치
```
- **데이터 버스**: 데이터 전송
- **주소 버스**: 메모리 주소 전송
- **제어 버스**: 제어 신호 전송

## 폰 노이만 구조
현대 컴퓨터의 기반이 되는 구조로, 프로그램과 데이터를 같은 메모리에 저장하는 방식입니다.
